Questão número 2

Descreva com suas palavras e entendimento o que é URL. Diferencie e exemplifique URL absoluta e URL relativa.

Resposta:

URL é o endereço de um determinado web site, podendo ser acessado via http/https ou IP. Quando você obtêm informação do endereço você pode acessá-lo de qualquer browser. A URL é constituida pelo Servidor, diretório e o arquivo no qual está sendo vizualisado. Unido com o Protocolo HTTP, eles mantém um padrão do que será acessado e se está ativo, autorizado ou impedido. As URLs Absolutas são endereços completos, incluindo protocolos, o domínio no qual o servidor está hospedado, sendo utilizado para apontar arquivos que pertencem a outro site. Ex: http://www.google.com As URLs relativas são curtas, omitindo partes da sintaxe do endereço, fazendo o browser ser o encarregado de "completar" com partes da página atual. Por isso são relativos, não há necessidade de sair do diretório pois o arquivo origem está no mesmo local do arquivo alvo, sendo usado para arquivos localizados no mesmo site. Ex: site.exemplo/index.html

Questão número 3

Antes da leitura da Aula 01, você sabia diferenciar os termos: home-page, página web e web site? Justifique sua resposta e ainda comente sobre cada um deles de forma a diferenciá-los.

Resposta:

Eu tinha um breve conhecimento sobre: home page, página web e web site por conta do contato com jogos e sites em inglês, mesmo com 8 anos. Não me gabando mas com essa idade eu já tinha conhecimento para onde queria ir. Como era muito curioso eu abria vários links e observava como a barra de enredeços, URL, modificava.

Questão número 4

Dentro do contexto web, pesquise, defina e descreva com suas palavras e entendimento sobre os termos: Front end e Back end.

Resposta:

Front-end é a parte visual do usuário. São um conjunto de ferramentas para que possa ser construido o layout, a página, que o usuário irá interagir, com JavaScript, HTML e CSS. É o papel do front-end facilitar o uso do Web Site ou Aplicação com o usuário final. Já no Back-end usasse mais a parte de lógica de programação, tratamento de dados recebidos e enviados, como serão armazenados no banco de dados. Utilizasse a linguagem Back-end como: Java, C#, C++, Python e entre outras. O mais interessante é como as duas técnologias se comunicam, o front-end e o back-end utilizam a API para se comunicar e trocar informações, front-end solicitando X informação, back-end recebe, trata a informação, confere no banco de dados e retorna para o front-end.

Aula 2

Questão número 1

Explique como ocorre o processo de relacionamento cliente/servidor web.

Resposta:

A comunicação entre cliente e servidor funciona da seguinte maneira:
  1. O usuário solicita as informações pelo URL, usando o protocolo http com o código resposta.
  2. O Servidor verifica se o protocolo obteve sucesso, não obteve sucesso ou não existe no servidor.
  3. Se houver resposta: De confirmação, erro ou inesistente, ele retorna para o usuário.
  4. A cópia do documento, web site, aparecerá na tela do usuário(Caso seja 200) ou irá aparecer que há um erro de servidor (Caso seja 404) ou cliente (Caso seja 500).
