Aula 7

FUNDAMENTOS EM CSS

Leia atentamente o guia de estudos e realize testes simulando as situações abaixo, e marque apenas as alternativas verdadeiras e justifique-as.


  1. a) Uma mesma folha de estilos pode ser usada por várias páginas.
  2. b) Uma mesma página pode usar várias folhas de estilo.
  3. c) Se um browser não suportar uma folha de estilos requerida pela página, poderá ha-ver uma degradação na qualidade da apresentação mas nunca haverá perda de in-formação.
  4. d) É possível construir um site inteiro usando apenas CSS.
  5. e) A linguagem CSS usada para construir folhas de estilo é uma recomendação do W3C – consórcio de empresas que estabelece os padrões para a Web.

Respostas:

  1. a – Está correto. O mesmo arquivo CSS pode ser usado por várias páginas contanto que em cada página seja chamada o arquivo para o do mesmo.

  2. b – Está correto mas por se tratar de CSS um atributo que estiver na página será o que prevalecerá, gerando confusão em futuras manutenções.

  3. c - Está correto. Se o site não suportar CSS apenas não irá mostrar as interações, cores e estilos da página, não afetando o conteúdo que a mesma possui(Implementado no HTML).

  4. d – Incorreto. CSS é apenas uma técnologia de estilização, não havendo conteúdo nenhum.

  5. e – Está correto. A W3C foi fundada com o objetito de padronizar as páginas web para que seja acessível a todas as pessoas, mesmo as que possuem dificuldades (As que necessítam de acessibilidade visual ou auditiva).
